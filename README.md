# Code Portfolio

Static single-page QA/automation portfolio.

## Files

- `index.html`: page structure
- `styles.css`: visual design and responsive layout
- `script.js`: editable project and stack data

## Customize

Edit the `projects` and `stack` arrays in `script.js` to replace the placeholder content with your real work.

## Run

Open `index.html` directly in a browser, or serve the folder with a simple static file server.

## Publish

### Vercel

This repo is ready for static deployment on Vercel. Import the folder or repo and deploy as a static site.

### Netlify

Deploy this folder directly with no build command and publish directory set to `.`

### GitHub Pages

Upload these files to a repo and serve from the repository root.
